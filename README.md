# Приложение для сравнивания эффективности разных раскладок клавиатур

Данный проект представляет собой программу для анализа различных русских клавиатурных раскладок.

## Рассмотренные раскладки
- **ЙЦУКЕН** — стандартная русская раскладка  
- **Вызов** — альтернативная раскладка  
- **Яверты** — модифицированная раскладка  

---

## Цель проекта
Сравнить эффективность раскладок по нагрузке на пальцы при 10‑пальцевой печати.  
Программа вычисляет штрафы за движения пальцев (вверх, вниз, вправо, влево, по диагонали) и строит графики нагрузок.

---

## Методика расчёта
- Простое движение на соседнюю клавишу: **1 балл**  
- Движение на два шага или диагональ: **2 балла**  
- Сложные случаи (например, буквы `Х` или `Ъ` в ЙЦУКЕН): **3–4 балла**  
- Итоговая нагрузка = сумма штрафов по каждому пальцу обеих рук

---

## Входные данные
1. Текст: роман *«Война и мир»* (без иностранных букв)  
2. Файл лексем: список словоформ русского языка  
3. Файл диграмм: `sortbukw.csv` (647 строк, отсортированные диграммы)
